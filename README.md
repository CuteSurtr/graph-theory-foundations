# Graph Theory: Foundations, Structures, and Applications

This is an independent project by Jiho Lee (UCSD) that aims to create a comprehensive guide to graph theory, combining mathematical rigor, visualization, algorithmic implementations, and practical exercises.

## Structure
- LaTeX source files: `/src/`
- Compiled PDF: `/output/`

## Status
Work in progress. Targeting ~20,000 lines by completion.

## Topics Covered
- Basic Classes of Graphs
- Eulerian and Hamiltonian Graphs
- De Bruijn Sequences
- Proofs, Algorithms, Exercises, and more.
